# SkyTrust无人机共享租赁区块链平台 - 项目架构说明

## 项目概述

已成功为SkyTrust无人机共享租赁区块链平台构建了完整的Spring Boot单体应用项目架构。该架构遵循分层设计原则，集成区块链、AI服务和物联网等核心能力。

## 关键文件说明

### 1. 主应用类
- `SkyTrustApplication.java` - Spring Boot主启动类，启用异步和定时任务支持

### 2. 配置文件
- `application.yml` - 主配置文件，定义通用配置
- `application-dev.yml` - 开发环境配置（本地数据库连接等）
- `application-prod.yml` - 生产环境配置（环境变量注入）

### 3. 数据库脚本
- `scripts/db/init.sql` - 完整的数据库初始化脚本，包含：
  - 用户表、设备表、租赁订单表、支付记录表
  - 区块链交易记录表、操作日志表
  - 视图、存储过程、函数和事件
  - 初始化数据插入

### 4. 部署脚本
- `scripts/deploy/deploy.sh` - 自动化部署脚本，支持：
  - 多环境部署（dev/test/prod）
  - Docker和Kubernetes部署
  - 健康检查、数据库迁移
  - 详细的日志和错误处理

### 5. Docker配置
- `Dockerfile` - 多阶段构建，优化生产镜像
- `docker-compose.yml` - 完整的本地开发环境，包括：
  - MySQL、Redis、RabbitMQ、MinIO
  - MQTT Broker、Ethereum节点（Ganache）
  - AI服务（Python Flask）、Spring Boot应用
  - 健康检查和依赖管理

## 快速启动指南

### 1. 本地开发
```bash
# 启动依赖服务
docker-compose up -d mysql redis rabbitmq minio mqtt ganache ai-service

# 运行应用
mvn spring-boot:run -Pdev
```

### 2. Docker部署
```bash
# 完整部署
docker-compose up -d

# 查看服务状态
docker-compose ps
```

### 3. 生产部署
```bash
# 使用部署脚本
./scripts/deploy/deploy.sh -e prod -b -d

# 或手动部署
docker build -t skytrust-platform:prod .
docker-compose -f docker-compose.prod.yml up -d
```

1. **安全配置**：生产环境必须修改JWT密钥、数据库密码等敏感信息
